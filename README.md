# Veiled

隐身 AI 助手客户端，从安装到运行全程伪装为 Windows 系统显示驱动辅助工具。

## 功能

- **悬浮对话窗** — 全局热键唤起，无边框半透明，不出现在任务栏和 Alt+Tab
- **多服务商 / 多模型** — 可创建任意多个服务商，每个有独立的接口地址、密钥、额外参数和模型列表；支持一键「获取模型列表」
- **对话内切换模型** — 顶部模型芯片随时切换服务商 / 模型，带 👁 的为视觉模型
- **更多菜单** — 对话右上角 `⋯` 菜单聚合了新建 / 历史 / 清空 / 删除 / 导出 / 翻译 / 总结 / 设置 / 帮助
- **截图问答** — 框选屏幕区域发送给 AI；选中视觉模型即可直接看图，否则可走「视觉识别中继」转写
- **剪贴板快捷问答** — 选中文字按热键，AI 回答以系统通知弹出
- **通知伪装** — 回答通知可伪装为 QQ / 微信 / Edge 消息
- **截屏保护** — 所有窗口对截屏和录屏不可见（`SetWindowDisplayAffinity`）
- **环境感知** — 检测到录屏、远程桌面、会议软件时自动静默
- **老板键** — 一键瞬间隐藏所有界面
- **侧边栏设置** — 服务商与模型 / 视觉识别 / 快捷键 / 外观 / 隐私 / 行为 / 关于
- **本地加密存储** — 对话记录和配置使用 Windows DPAPI 加密

## 快速开始

```bash
pip install -r requirements.txt
python run.py
```

首次运行会弹出初始化向导，配置 API Key 和快捷键后程序进入后台。

## 默认快捷键

| 功能 | 快捷键 |
|------|--------|
| 唤起/隐藏对话窗 | `Ctrl+Shift+Space` |
| 老板键 | `` Ctrl+` `` |
| 剪贴板问答 | `Ctrl+Shift+Q` |
| 截图问答 | `Ctrl+Shift+S` |
| 退出程序 | `Ctrl+Shift+Alt+Q` |

## 对话内功能

顶部「模型芯片」可切换服务商 / 模型；右上角 `⋯` 菜单提供下列全部功能，也可直接输入斜杠命令：

| 指令 | 功能 |
|------|------|
| `/new` | 新建对话 |
| `/list` | 历史对话 |
| `/clear` | 清除当前对话 |
| `/delete` | 删除当前对话 |
| `/export` | 导出对话 |
| `/t <文字>` | 翻译（留空则翻译剪贴板） |
| `/s` | 总结剪贴板内容 |
| `/model` | 显示当前模型并提示如何切换 |
| `/settings` | 打开设置 |
| `/help` | 帮助 |

## 服务商与模型

在「设置 → 服务商与模型」中可：

- 新增 / 删除任意数量的服务商，分别设置名称、协议（OpenAI 兼容 / Claude）、Endpoint、API Key、额外请求参数（JSON）
- 为每个服务商维护模型列表：手动添加，或「获取列表」从接口拉取；可逐个标记是否为**视觉模型**、设为**默认模型**
- 设置一个共用的全局代理
- 旧版配置（单一 provider 字段）会在首次启动时自动迁移为新结构

## 打包

```bash
pyinstaller build.spec
```

输出 `DAHService.exe`，伪装为 Windows Display Adapter Helper Service。
